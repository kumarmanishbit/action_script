# action_script

### Repository events:-

    - push
    - pull_request
    - release

### Webhook

    - Branch creation
    - Issues
    - Memebers joined

### Scheduled

    - cron format

### Jobs

    - Workflows must have at least one job.

### runs-on:

    - Environment where we want our workflow to run

### steps:

    - List of actions or commands.
    - Each step runs in its own process.

### uses:

    - Identifies an action to use
    - Defines the location of that action
